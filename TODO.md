# ZenScreen 优化待办事项 (TODO)

本项目已完成核心功能开发与品牌重塑，以下是建议的后续高性价比优化方向：

## 1. 用户体验 (UX) 🚀
- [ ] **鼠标光标增强**：在编辑器中增加“光标高亮”滤镜（在 `frameRenderer.ts` 中实现）。
- [ ] **社交媒体预设**：针对 **B站 (16:9)**、**抖音 (9:16)**、**小红书 (3:4/9:16)** 等国内主流平台提供快速剪裁与导出预设。
- [ ] **编辑自动保存**：实现每 30s 自动保存 `.zenscreen` 项目状态，防止意外崩溃导致丢进度。
- [ ] **导出后动作**：视频导出完成后自动在资源管理器中打开目标文件夹。

## 2. 性能与资源 (Performance) ⚡
- [ ] **码率档位选择**：在录制设置中提供“均衡”、“高画质”、“原画”三个码率档位。
- [ ] **资源预加载**：使用 `PIXI.Cache` 预加载编辑器常用的壁纸和 UI 贴图，消除切换卡顿。
- [ ] **多线程导出**：将视频解码与渲染循环迁移至 Web Worker，彻底解放 UI 主线程。

## 3. 工程与维护 (Engineering) 🛠️

- [ ] **核心算法测试**：为 `compositeLayout.ts` 和录制时长修正逻辑增加更全的 Vitest 单元测试用例。

## 4. 辅助功能 (Other) ⚙️
- [ ] **热键支持扩展**：为批注组件（文本、箭头、画笔）增加快捷键切换支持。
- [ ] **硬件加速检测**：在设置界面清晰显示当前系统是否启用了 GPU 编码加速。
