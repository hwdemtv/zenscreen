# ZenScreen 使用说明书 📖

欢迎使用 **ZenScreen**！这是一个旨在提升视频演示质量的禅系录屏编辑器。

---

## 1. 录制阶段 (Recording) ⏺️

### 1.1 选择并录制
1. 启动应用后，点击 **"选择录制源"**。
2. 在弹出窗口中选择要录制的 **整个屏幕** 或 **特定窗口**。
3. 配置 **麦克风** 和 **系统音频** 开关。
4. 点击红色 **录制按钮** (或使用快捷键) 开始。

### 1.2 结束录制
录制完成后，点击悬浮窗上的 **停止按钮**。应用会自动进入编辑界面。

---

## 2. 编辑阶段 (Editing) ✂️

### 2.1 缩放控制 (Zoom) 🔍
- **自动缩放**: 系统会根据录制时的鼠标点击自动生成缩放建议。
- **手动缩放**: 
    - 在时间轴上右键点击，选择“添加缩放关键帧”。
    - 拖动预览框中的矩形区域来调整缩放位置和范围。
- **平滑度**: 默认开启 **运动模糊 (Motion Blur)**，让缩放过程看起来非常丝滑。

### 2.2 剪辑与修整 (Trimming) ✂️
- **添加剪辑**: 
    - 在时间轴上将指针移动到目标位置。
    - 按下 **`T` 键** 或点击侧边栏的 **剪刀图标**。
    - 拖动生成的 **红色区域** 边缘，覆盖你想要“剪掉”的部分。
- **注意**: 被红色区域覆盖的内容在导出时将完全消失。

### 2.3 视觉美化 (Styling) 🎨
- **背景**: 在右侧面板中，你可以为视频添加：
    - 各种精美的桌面壁纸。
    - 纯色或渐变背景。
- **布局**: 可以调整视频的 **内边距 (Padding)** 和 **圆角 (Corner Radius)**，让画面更具现代感。

### 2.4 批注与说明 (Annotations) ✍️
- 点击工具栏的 **文字** 或 **箭头** 图标。
- 在视频画面上直接点击并输入/调整位置。
- 批注会在时间轴上显示为独立的图层，可以调整其显隐时间。

---

## 3. 导出与分享 (Export) 📤

1. 点击右上角的 **"导出"** 按钮。
2. **选择预设**: 
    - **1080p (YouTube/B站)**
    - **9:16 (抖音/小红书)**
3. **格式选择**: 推荐使用 **MP4**（兼容性最好），轻量演示可选择 **GIF**。
4. **高性能**: ZenScreen 使用 WebCodecs 原生加速，导出速度极快。

---

## 4. 快捷键指南 ⌨️

| 快捷键 | 功能说明 |
| :--- | :--- |
| **`Space`** | 播放 / 暂停 |
| **`T`** | 在当前位置添加剪辑区域 (Trim) |
| **`Z`** | 在当前位置添加缩放点 |
| **`Delete`** | 删除选中的剪辑或批注 |
| **`Ctrl+S`** | 保存项目文件 (`.zenscreen`) |
| **`Ctrl+Z` / `Ctrl+Y`** | 撤销 / 重做 |

---

## 5. 常见问题 (FAQ) ❓

- **找不到录制的文件？**
    录制后的原始素材保存在应用的 `userData/recordings` 目录下。
- **系统声音录不进去？**
    - Windows 用户通常直接可用。
    - macOS 用户请务必在系统提示时允许“屏幕音频捕捉”权限。
- **导出的视频黑屏？**
    请确保在设置中开启了“硬件加速”，并检查显卡驱动。
